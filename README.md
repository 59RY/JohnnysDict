ジャニーズ用語辞書

## これは何？

ジャニーズに関する用語辞書データです。


## 登録単語リスト
- エクセルで見る：**[登録単語.xlsb](/登録単語.xlsb)**
- テキストで見る：**[登録単語.csv](/登録単語.csv)**
